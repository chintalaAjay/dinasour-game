# Dinosaur Game 🦖

A fun and interactive Dinosaur Runner Game built using HTML, CSS, and JavaScript.

The player controls a dinosaur and avoids obstacles to achieve the highest score possible.

---

## 📌 Features

- 🎮 Simple and interactive gameplay
- 🦖 Dinosaur jump animation
- 🐉 Moving obstacle/dragon
- 🔊 Background music and game over sound effects
- 📱 Responsive design
- 🚀 Hosted using Firebase Hosting

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase Hosting

---

## ⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/chintalaAjay/dinasour-game.git
```

2. Open the project folder

```bash
cd dinasour-game
```

3. Open the project in VS Code

4. Install the **Live Server** extension in VS Code

5. Right-click on `index.html`

6. Click **Open with Live Server**
